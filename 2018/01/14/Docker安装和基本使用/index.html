Docker安装和基本使用

发表于 2018-01-14 | 分类于 Docker

本文记录mac下安装docker的过程，以及docker的简单使用。

系统要求

Docker for Mac要求最低系统版本为10.10.3 Yosemite，如不满足最低要求，可以安装Docker Toolbox。

安装

使用Homebrew安装

Homebrew的cask已经支持Docker for mac，安装有Homebrew的用户直接在终端执行以下命令就能安装：

下载安装

直接点此下载安装镜像，最好选Stable版本的的，双击安装即可，期间需要输入管理员密码。

运行

运行后系统状态栏会出现一个鲸鱼图标，表明docker服务端已经运行起来了。

此时在终端执行docker version或docker info可以查看docker版本等信息。

加速

docker默认从海外的docker hub加载镜像，速度会非常慢，建议使用国内镜像。设置方式为状态栏小图标 -> Preferences... -> Daemon -> Registry mirrors: -> + -> http://hub-mirror.c.163.com -> Apply & Restart。

Hello World

在终端执行 docker run library/hello-world，docker检测本地是否有hello-word镜像，如果有就直接运行，没有则会从官方的镜像库library中拉取hello world镜像到本地然后运行。由于官方镜像库名称固定为library，所以也可以缺省，即从官方库拉镜像可用 docker pull 镜像名命令。
